
# ToyRobot


## Dependencies

```
gem "simplecov", "~> 0.22.0"
gem "rspec", "~> 3.12"
```

## Preparation

```shell
bundle install
```

## Playing with the Robot

```shell
./play_with_robot.rb
```

## Running Tests

```shell
rspec --format=documentation toy_robot_spec.rb
```

## Test Coverage

```shell
open ./coverage/index.html
```
